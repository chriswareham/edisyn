Waldorf MicroWave II / XT / XTk

Multi Patch Editor

This editor will work with the Waldorf MicroWave II, the Waldorf MicroWave XT, and the Waldorf MicroWave XTk in 10 voice or in 30 voice.

Communicating with Edisyn

Set your MicroWave to send Sysex only, not Ctl nor Ctl+Sysex. Also set Receive to on.

The Show Button

Each Instrument has a button called Show. This shows the given sound (bank and number) as a separate Blofeld Single editor panel.

XTk Options

Two options are only available for the MicroWave XTk:
  • MIDI Send (In General) Set this to Global to send MIDI on all instruments regardless of their individual settings. Set this to Specific to allow individual instruments to specify whether they want their MIDI sent.

  • MIDI Send (In each Instrument) determines whether a given instrument will send its MIDI out. This is overridden by the General option above.


Gotchas

There are a few:
  • Edisyn cannot tell the MicroWave to switch to a given multi patch number via a program change. As a result, when you request a patch, Edisyn will receive the patch but the MicroWave will continue to play the old patch. Solution: after receiving the patch, issue Send To Current Patch to update the MicroWave. For a similar reason, Send To Patch is disabled.

  • Though the MicroWave responds to individual parameter changes in Multimode, it does not send any parameter change sysex when you turn its knobs.

  • Also, when you make individual parameter changes, the MicroWave's screen won't update to reflect those changes: but they've been made. Try scrolling away and then back, and you'll see the changes magically appear.


Librarian Support

Edisyn can request all the patches from the II/XT/XTk at the same time via the menu Request All Patches from Synth. It's a lot faster than the standard download methods.

Fun Feature

When your patch editor becomes the front window, it'll ask the MicroWave to change to Multi mode automatically.
