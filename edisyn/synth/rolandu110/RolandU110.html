Roland U-110

Patch Editor



Communicating with Edisyn

Set Edisyn's MIDI channel to the same as the synth's Control Channel. Turn ON Program Change and Exclusive.

Gotchas

  • The U-110 doesn't seem to respond to requests to download individual patches: so Edisyn must change the patch to the given patch (loading it into current memory) and then request the patch from current memory.

  • Incredibly, Roland's sysex specification is missing almost everything needed to write the editor, so much had to be reverse engineered. If you'd like to see a working specification, see the comments at the end of Edisyn's RolandU110.java file.


Librarian Support

Edisyn's U-110 editor (unlike the U-220 editors) supports bulk downloads from the machine, and in fact has a dedicated menu option in the Librarian to ask the machine to send it a bulk dump, which is much faster than Edisyn asking for each patch one by one. Try it out, it's called Request Bank from Synth.
