Novation A-Station

Patch Editor - Still under development

This is an editor for the Novation A-Station.
It is developed and tested with the latest available firmware, being '2.1.2'

Communicating with Edisyn
Set Edisyn's MIDI channel to the same as your synth's MIDI channel

Other Interesting Facts
