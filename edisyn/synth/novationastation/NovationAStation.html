Novation A-Station

Patch Editor - Still under development !

This editor will work with the Novation A-Station.
It is developed and tested with the latest available firmware, being '2.1.2'

Communicating with Edisyn
Set Edisyn's MIDI channel to the same as your synth's MIDI channel

Other Interesting Facts
This editor is still under development and not feature complete.
On the currently-supported-features list we have:

  • request current/given patch, supporting most of the parameters
  • most of the realtime controls (CC-based) are working bidirectional
  • NRPN-based controls are still under development
Although the UI is still limited, it's already more useful than the UI on the device ;).

On the TODO list we have (in order of importance - IMHO):
  • writing updates to the device
  • supporting NRPN-based controls
  • testing, testing, testing... (the model + UI, but also interaction with global Edisyn features: merges, librarian, ...)
  • UI extensions (supporting the whole set of parameters)
  • UI improvement (some ideas on usability)
