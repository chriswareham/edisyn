Novation A Station

Patch Editor

This is an editor for the Novation A Station.
It is developed and tested with the latest available firmware, being "2.1.2"

Communicating with Edisyn
Set Edisyn's MIDI channel to the same as your synth's MIDI channel. If you want to store patches, make sure the "Global Memory Protect" is OFF on the synth.

Notes
The A Station editor supports all basic patch + parameter handling:

  • supporting realtime control (both directions)
  • supporting all parameters (from both the hardware controls and the menu's)
  • load current/given/next patch
  • write to current/given path
  • basic librarian support

Gotchas

  • If you load the current patch, the A Station does not tell Edisyn what bank or number the patch is.

  • Some parameters (LFO, delay, ...) have both a Dial for sync and non-sync speed. On the unit, the sync speed has precedence, so only if its value is "N/A" the non-sync speed is being applied To make that explicit in the editor the non-sync speed dial is disabled when the sync speed dial is NOT "N/A"
