Novation A-Station

Patch Editor - Initial version

This is an editor for the Novation A-Station.
It is developed and tested with the latest available firmware, being '2.1.2

Communicating with Edisyn

  • Set Edisyn's MIDI channel to the same as your synth's MIDI channel.
  • If you want to store patches, make sure the "Global Memory Protect" is OFF on the synth.

Other Interesting Facts
The current version supports all basic patch + parameter handling:

  • supporting all parameters (from both the hardware knobs/switches, but also the menu's)
  • supporting realtime control (both directions)
  • load current/given/next patch
  • write to current/given path
