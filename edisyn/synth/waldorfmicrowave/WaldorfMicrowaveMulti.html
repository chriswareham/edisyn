Waldorf MicroWave

Single Patch Editor

This editor will work with the Waldorf MicroWave (sometimes called the "Microwave 1") only. The MicroWave II/XT/XTk have a different editor.

Communicating with Edisyn



Gotchas

  • This one is important: Edisyn cannot change the MicroWave from Single mode to Multimode or back via sysex. You have to do it manually on the unit. This has a big impact when Edisyn is writing or requesting patches, because when it does so it also ask to change the patch, but this will not happen if the MicroWave is set to the wrong mode.
